💳 Sistema de Gestión de Créditos y Cartera
Este proyecto consiste en una aplicación integral diseñada para la administración, seguimiento y recaudo de carteras de crédito en tiempo real. La plataforma permite optimizar la operación de empresas que realizan sus prestamos, facilitando el control detallado de los abonos, estados de cuenta y rendimiento financiero diario donde son atendidos por sus asesores.

🚀 Tecnologías Utilizadas (Stack PERN)
El sistema fue desarrollado utilizando el stack PERN, garantizando una arquitectura robusta, escalable y eficiente:

PostgreSQL: Base de datos relacional para la gestión persistente de clientes y movimientos financieros, incluyendo el uso de restricciones únicas para integridad de datos.

Express: Framework de servidor para Node.js encargado de la lógica de negocio y la API REST.

React: Biblioteca de frontend para crear una interfaz de usuario dinámica, responsiva y optimizada para dispositivos móviles.

Node.js: Entorno de ejecución para el backend, proporcionando una comunicación rápida y fluida.

🛠️ Funcionalidades Principales
Módulo de Cobrador: Interfaz optimizada para móviles que permite a los cobradores gestionar sus rutas, visualizar clientes y registrar abonos al instante.

Integración con WhatsApp: Redirección directa desde el perfil del cliente para facilitar la comunicación y el cobro preventivo.

Monitoreo en Tiempo Real: Seguimiento de recaudo diario por usuario, permitiendo un control financiero preciso.

Gestión de Cartera: Sistema de filtrado avanzado por cliente, cédula o número de crédito, y manejo automático de estados (Vigente, Mora, etc.).

Dashboard de Rendimiento: Generación de reportes y visualización de la salud de la cartera para la toma de decisiones.

📖 Propósito del Sistema
Este software nació de la necesidad de digitalizar y profesionalizar el sector de microcréditos. Su objetivo principal es:

Reducir el error humano en el registro manual de pagos.

Optimizar los tiempos de cobro mediante herramientas digitales.

Garantizar la transparencia entre el administrador, el cobrador y el cliente final a través de reportes detallados y comunicación directa.
